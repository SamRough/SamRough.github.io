The Important Thing

做正确的事，如果发现自己在做不正确的事情，什么时候停下来都不晚。

核心问题，你想要过怎样的人生：自由、幸福、平和、健康、自信、有活力、富足

基于此人生目标，真正重要的事情是什么？做对的事情，不对的事情不做。也就是能帮助实现目标的事情就做，没有帮助的就不做。方向比努力更重要，所以方向是什么，当下做哪些事情可以帮助我实现以上目标？

基本事实和原则：

  1. 做对的事情，不对的事情不做，方向比努力重要。
  2. 理清思路很重要，脑子混乱容易被淹没在琐事当中，只是让人分心，渐渐远离目标。
  3. 跑步是少数可以帮助大脑理清思路的活动。
  4. 财富观：
    1. 金钱很重要，财富自由是人生自由的基础。
    2. 财富自由的时间也很重要，30岁的财富自由比65-70岁退休的财富自由有意义的多。早九晚五的上班+理性投资，也能财富自由就是很慢，三四十年的辛勤工作熬到退休也能财富自由，但命不久矣。加入创业公司成为股东，同样都是早九晚五，但是很快。这里面本质的区别就是财富自由的时间。一个快一个慢，财富自由这件事上快比慢好。
    3. 超额收益来自复利，复利的核心是体量大小和持续的时间，雪球尽量大，时间尽量长。
    4. 为了持续滚动，我们需要赚钱不亏钱。芒格说的棒球理论，只打好球，不打差球。就算可能三阵出局也不打差球。保持耐心等待好球出现，奋力一击打出全垒打。
    5. 加杠杆是为了让雪球足够大，雪球确实越大越好，但杠杆不是越高越好。
    6. 杠杆最大的风险是会让雪球归零，甚至欠钱。智者永远不欠钱，因为负债也会滚雪球。
    7. 杠杆不一定是借贷，很多事情自带杠杆属性，比如：视频、书籍、代码、音乐、利滚利
    8. 借贷和欠钱是两码事，借贷后资产负债可能为正数，欠钱是资不抵债资产为负。总结一句话，无论如何不能冒着资产为负的风险。
    9. 保持兴趣持续学习，一是为了识别好球坏球，二是为了能长期的持有。长期持有不是为了长期而长期，主要目的是长期持有产生复利。
    10. 现在这个环境下，赚美金比赚人民币好。赚美金的方式：投资美股、在美国公司工作拿美国薪水、做美国人生意。可以全做，也可以一项做精。
    11. 判断清楚的投资比不投资要好，不投资比乱投资要好。
  5. 家庭观：
    1. 家和万事兴，和睦温馨的家庭是人生自由的基础。
    2. 投资自己胜过服从伴侣，家庭需要经营，而不是无条件迁就。
    3. 好的外表有注意家庭合睦。
    4. 健康是家庭合睦的关键。
    5. 保险真的可以保太平。
    6. 自己变得更好，会潜移默化的让伴侣变得更好。
    7. 私人空间有助于家庭和睦，不要随时随地粘在一起。
  6. 健康观：
    1. 运动使人健康，不只是生理上，还有心理上。
    2. 一件事情该不该做，要有短期和长期的观念。建立一种用长期主义思考的习惯，不用短期愉快，换长期不不愉快。酗酒、抽烟、各种放纵（饮食、社交媒体、刷短视频、刷长视频等）
      1. 一件事情，如果当前做很不愉快，但长期有很大好处。那就要做。
      2. 一件事情，如果当前做很愉快，但长期有很大坏处。那就绝不要做。
    3. 跑步是少数有助于清理头脑，理清思路的运动。妙用无穷。
    4. 睡眠的重要性应在第一优先级，睡不够的危害被低估了。尤其是头脑混乱，情绪会低落。尤其会影响判断能力，进一步影响成就。
    5. 学习佛理，行善积德有助于增长智慧，身心也会更健康。
  7. 事业观：
    1. 工作的目的是为了获得报酬，报酬高低主要由市场供需关系主导，稀缺价高泛滥价低。所以能力要有稀缺性，才是核心竞争力。能力可以是：个人能力、专业能力、个人关系、权利背景、家庭背景、金钱、消息等等，所有可交换物都是能力，而不仅仅指个人能力和专业能力。
    2. 工作环境不应该压抑和影响心情，不应影响健康。
    3. 工作首要考虑成长性（这里成长性是指能否长见识丰富价值观，提升个人稀缺性。），这里的技巧是在有前景的行业做有前景的工作。
    4. Title不应该成为追求的目标，稀缺性提升和报酬提升才是追求的目标。
    5. 人要有事做，工作是有必要的，闲的发慌甚至会影响健康。（投资不是这样）
    6. 在没有进入复利魔力前，工作除养家糊口外，核心目标是托举你，直到进入复利。
    7. 如果是早九晚五，工作时间只占全部时间的18%，不应为18%消耗过多精力和情绪。而且18%可以提供100%的生活报酬，性价比还是高的。
    8. 语言能力是一种特殊能力，掌握它具有极高的性价比，重要程度不亚于睡眠和身体健康。它会影响你的价值观，判断，情绪。你见过哪个聋哑人士成为政治家、商业领袖、创业家、思想家？

基于以上认知，我应该把主要精力放在哪里去实现目标？

  1. 投资美股
  2. 持续工作努力获得稀缺性，跳槽获取更大报酬。（Esunny项目好好做，跳槽也好好看。）
  3. 保持身心健康头脑清楚：保障睡眠、干净饮食、规律运动。（短期痛苦，长期巨大回报）
  4. 语言：粤语和英文，粤语说和听的能力，英文写的能力。（短期痛苦，长期巨大回报）

基于以上规划，每日的主要工作是什么？

  1. 持续投资美股
  2. 好好工作，Esunny项目（稀缺性）
  3. 学习AI，准备跳槽（稀缺性）
  4. 练习粤语口语和英文写作，注意是练习而不是学习！（稀缺性）
  5. 跑步、早睡早起、干净饮食减重（短期痛苦，长期受益）
